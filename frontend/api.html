BondCreate Escrow

API Reference

Integrate Bond escrow into any marketplace, Discord bot, or Reddit community.

Base URL: http://localhost:3001
  • Create Escrow
  • Get Transaction
  • Simulate Payment
  • Mark Delivery
  • Confirm Receipt
  • Raise Dispute
  • Resolve Dispute
  • Live Updates (SSE)
POST
/api/escrow

Create a new escrow transaction. Returns the escrow ID and a shareable link.

{
  "itemName": "MacBook Pro",
  "description": "2023 model, M3",
  "amountINR": 120000,
  "sellerUpiId": "seller@upi",
  "buyerEmail": "buyer@email.com",
  "expiryHours": 48
}
GET
/api/escrow/:id

Fetch full transaction details including state, history, and parties.

POST
/api/escrow/:id/pay

Simulate a UPI payment. Transitions state CREATED → FUNDED.

POST
/api/escrow/:id/deliver

Seller confirms delivery. Transitions FUNDED → DELIVERED.

{ "deliveryNote": "Shipped via Delhivery, tracking #XYZ" }
POST
/api/escrow/:id/confirm

Buyer confirms receipt. Auto-progresses DELIVERED → CONFIRMED → RELEASED and triggers fund release.

POST
/api/escrow/:id/dispute

Buyer raises a dispute. Transitions DELIVERED → DISPUTED. Funds remain locked.

{ "disputeReason": "Item not as described" }
POST
/api/escrow/:id/resolve

Admin resolves a dispute (mocked). DISPUTED → RESOLVED.

GET
/api/escrow/:id/stream

Server-Sent Events stream for live transaction state updates. Connect from the frontend to get real-time state changes.

const es = new EventSource('/api/escrow/TXN123/stream');
es.onmessage = (e) => {
  const { type, transaction } = JSON.parse(e.data);
  console.log(transaction.state); // 'FUNDED', 'DELIVERED', etc.
};
